## 💪 아쉬웠던 점 & 느낀점
- 프로젝트 경험이 없다보니 기획부터 설계까지 어떻게 해야할지 감조차 잡히질 않아 시간을 많이 허비했었다.
- 기획에서 부족하다보니 완성도가 많이 떨어질 수 밖에 없었다.
- 서툴다 보니 반응형을 완벽히 구현하지 못 한 부분이 아쉬웠다.
- 첫 프로젝트를 아쉽게 마무리했는데, 이번에 구현해내지 못한 부분들을 다른 프로젝트때 보완해서 구현하도록 노력해야겠다!
